# FRB candidate viewer

This code can be used for plotting FRB candidates. This is fredda verison.

### Prerequisites

```
pandas
bokeh 0.12.6
sigpyproc
mbplotlib
```

### Deploying

Simply deploy on your system and create a ~/.candplotter.cfg file based on the included example and run a bokeh server.
```
bokeh serve main.py
```
